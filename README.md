# ☕ Estudos de Java - FATEC Franca

Este repositório foi criado para organizar e documentar minha evolução na linguagem **Java** durante o curso de Análise e Desenvolvimento de Sistemas (ADS).

## 📌 Sobre o Repositório
Aqui registro exercícios de lógica, algoritmos e projetos práticos desenvolvidos em aula. O objetivo é consolidar os fundamentos da programação orientada a objetos e estruturas de dados.

## 🚀 Tecnologias Utilizadas
* **Java SE 25 (LTS)** 
* **JDK 25** 
* **IDE:** Visual Studio Code

## 📂 Estrutura de Exercícios

O repositório está organizado por tópicos evolutivos, acompanhando a grade da FATEC:

* **`Exercicio_JOptionPane`**: Introdução à interface gráfica simples para entrada e saída de dados.
* **`Exercicio_Sequenciais`**: Exercícios focados em lógica linear e operadores matemáticos.
* **`Exercicios_Decisao`**: Práticas com estruturas de controle (`if/else`, `switch`).
* **`Exercicios_String`**: Manipulação e tratamento de textos (Strings).
* **`Exercicios_Funcoes`**: Organização de código através de Procedimentos e Funções.
* **`Exercicios_Vetores+Try-Catch-Finally`**: Estruturas de dados (arrays) e tratamento de exceções para garantir a robustez do software.
* **`Exercicios_ArqTEXT`**: Persistência de dados através da leitura e escrita em arquivos de texto.
* **`Exercícios_Pacotes`**: Organização e modularização de projetos Java.
